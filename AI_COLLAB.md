# AI Collaboration Setup

## Opencode
- Used for repo management, multi-agent workflows, and code review
- Install: curl -fsSL https://opencode.ai/install | bash
- Use Gemini as backend for Android-specific tasks

## Gemini
- Used in Android Studio for code generation, UI, and testing
- Activate in Android Studio settings

## Workflow
- Use Opencode for planning, code, and review
- Use Gemini for Android code, UI, and tests
- Share session links for collaboration
