# Cyanogenmod Reborn: Feature Plan

## Modern Features
- Advanced customization (themes, icons, status bar, lock screen)
- Privacy Guard (fine-grained app permissions)
- Performance enhancements (CPU/GPU tuning, battery optimizations)
- Frequent OTA updates
- Native FLAC, OpenVPN, advanced reboot, built-in tethering
- Custom notification toggles
- Easy root access, unlockable bootloader
- Modern device support (Android 14+)
- Minimal bloatware
